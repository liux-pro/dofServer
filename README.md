### 快速使用

下载dof.sh
wget https://raw.githubusercontent.com/nnn149/dofServer/main/dof.sh
vi dof.sh 可以修改一些自定义配置(外放访问请改ip)
chmod 777 dof.sh
./dof.sh

选择4和9安装镜像
最后选择1进入容器按提示操作
